Executive Summary

Hyperio developed a customized web-based platform

The web application was designed for pharmacies in Zurich to manage COVID-19 testing and vaccination appointments for patients. The platform streamlined the booking process, reduced physical exposure for patients and pharmacists, ensured compliance with government guidelines, and provided automatic transmission and reporting of COVID cases to the government. Over a thousand patients used the platform during the crisis period of 2022.

Bootstrap, PHP, MySQL

Technological Design

Because the application needed mobile responsiveness, Bootstrap was an obvious choice for the front-end technology. It is fast to use and has a complete set of standardized form components - perfect for our registration platform.

Built with reliable and time-tested frameworks

For the Backend, we used a LAMP environment, including PHP and MySQL

The essential advantage of this programming language is its compatibility with SQL databases. We were able to write the support for the subscription module quickly, the booking system and the payment transactions thanks to PHP modularity.

A flexible and fast Backend

PHP is also handy when it comes to writing cURL queries thanks to its integrated curl_ library. Since we used PayPal for the payment module, we needed to issue API queries on their URL, and PHP let us quickly use the PayPal REST API.

A user-friendly Frontend

The platform was designed to be user-friendly and secure, offering patients an ergonomic, fast, and secure experience. For pharmacies, it improved the scheduling and administrative management of COVID services to patients.
